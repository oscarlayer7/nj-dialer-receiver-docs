---
title: "Colgar llamada"
openapi: "POST /api/hangup"
description: "Finaliza y cuelga una llamada activa."
---

## Descripción

Termina la llamada de forma inmediata. Una vez colgada, el `call_id` ya no puede usarse para enviar nuevas acciones.

<Warning>
  Esta acción es **irreversible**. Una vez colgada la llamada, no puede reanudarse. Si solo quieres interrumpir la acción en curso sin colgar, usa [`/break`](/api-reference/break).
</Warning>

## Cuándo usarlo

- Al finalizar el flujo de la llamada normalmente
- Para cortar la llamada ante una condición de error en tu sistema
- Cuando el destinatario no es el contacto esperado

## Ejemplos

<CodeGroup>

```bash cURL
curl -X POST https://nj-dialer-receiver.rd.l7labs.art/api/hangup \
  -H "Authorization: Bearer <tu_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "call_id": "01JRX5Y2K3GHIJKL"
  }'
```

```javascript Node.js
await fetch('https://nj-dialer-receiver.rd.l7labs.art/api/hangup', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <tu_token>',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ call_id: '01JRX5Y2K3GHIJKL' }),
});
```

```php PHP
Http::withToken('<tu_token>')
    ->post('https://nj-dialer-receiver.rd.l7labs.art/api/hangup', [
        'call_id' => '01JRX5Y2K3GHIJKL',
    ]);
```

</CodeGroup>

## Respuesta

```json
{
  "request_id": "01JRX5Y2K3ABCDEF",
  "status": "queued"
}
```
